# 🚀 2D Space Shooter Game

Hi there, I would like to introduce about a simple 2D Space Shooter game developed using Unity.
This project was developed based on the "Complete C# Unity 2D Game Development (Updated To Unity 6)" course on Udemy, with additional custom features implemented by me.

## 🎮 Gameplay
- Control a spaceship and destroy incoming aliens
- Try to survive and manage your health by avoid alien attacks as long as possible
- Achieve score by shooting down aliens

## 🛠️ Features
- Player movement (keyboard input)
- Shooting system (keyboard + mouse input)
- Enemy spawning system
- Health (HP) system with UI bar
- Collision and damage handling
- Sound effects
- Basic animation (Charge shooting)
- Background music

## 💻 Technologies used
- Unity (version: 6.3)
- Progamming language: C#
- Unity Input System
- Git & Github
